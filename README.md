### 2. 操作指引

`(A)人員與上課情形：選擇將剛剛的 CSV 檔案，按開始執行。`

![](tutorial/a11.png)

### 3. 介面說明

- 點選學員 ID 可以觀看其環保課程的修課紀錄。
- 以顏色區分尚未修課、修一堂、修兩堂、修三堂以上的學員。
- 可將結果下載成 CSV 檔案，用 excel 開啟。

![](tutorial/a12.png)

`點選「下載 CSV 檔」"即可下載，用 excel 開啟後如下圖。`
