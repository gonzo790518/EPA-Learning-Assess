1.1 人員與上課情形資料表

請確認只用到第一個工作表的前兩個欄位，第一個欄位為ＩＤ，第二個欄位為該ＩＤ所修的課程。

另存新檔，存檔類型選擇 "CSV"。

選擇確定。

關閉檔案時，選擇不要儲存。

1.2 單位人員名單資料表

請確認只用到第一個工作表的第一個欄位，第一個欄位為ＩＤ，其餘欄位皆空白。

存檔類型請選擇 "CSV"，詳細步驟請參考上方 "1.1 人員與上課情形資料表" 部分。

2. 操作指引

若不需限制人員名單，請在 (A)人員與上課情形 欄位選擇 人員與上課情形資料表，按開始執行。

若需限制人員名單，請在 (A)人員與上課情形 欄位選擇 人員與上課情形資料表，在 (B) 單位人員名單 欄位選擇 單位人員名單資料表，按開始執行。

3. 介面說明

  • 點選學員 ID 可以觀看其環保課程的修課紀錄。
  • 以顏色區分尚未修課、修一堂、修兩堂、修三堂以上的學員。
  • 可將結果下載成 CSV 檔案，用 excel 開啟。
